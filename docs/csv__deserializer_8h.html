◆ csv_deserialize()

size_t csv_deserialize ( void *  context,
void *  ptr_to_struct,
CSV_DESERIALIZE_DESC *  desc,
size_t  stride,
CSV_PARSER *  parser,
size_t  no_of_records 
)

Deserializes the required number of records from the CSV buffer and places in the input buffer.

Parameters
contextThe user variable that is passed to the CSV_Deserializer from desc
ptr_to_structThe buffer where the data gets placed
descThe layout of the buffer. The desc->length MUST be equal to parser->columns
strideThe value added to ptr_to_struct to get to the next place where the next data is to be placed
parserThe parser from which the CSV is to be deserialized
no_of_recordsTotal number of records to deserialize
Returns
Number of records actually deserialized
